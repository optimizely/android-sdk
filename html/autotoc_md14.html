OPTLYJSONModel - Magical Data Modeling Framework for JSON

OPTLYJSONModel allows rapid creation of smart data models. You can use it in your iOS, macOS, watchOS and tvOS apps. Automatic introspection of your model classes and JSON input drastically reduces the amount of code you have to write.

OPTLYJSONModel is a modified copy of the original JSONModel covered by the "LICENSE" file mentioned at the end of this document.
